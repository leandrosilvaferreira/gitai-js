# Run tests

1. Find the canonical **test** command in the root `CLAUDE.md` (or `.claude/rules/`).
2. Run that exact command from the project root.
3. Report the actual output — pass/fail counts and any failures.
4. Never claim tests pass without having run them and seen the output.

If no test command is configured, say so and suggest adding one rather than guessing.
